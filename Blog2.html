Building a Personal Finance Tracker

Posted on March 22, 2025 by Violet Talemwa

For one of my portfolio projects, I decided to tackle a real-world problem: managing personal finances. I wanted to build a simple yet powerful finance tracker that helps users stay in control of their income and expenses.

How I Built It

I used HTML and CSS to structure and style the app, and JavaScript to handle user input, balance updates, and local storage integration. All data is saved locally in the browser, so users can revisit their info without logging in.

Extra Features

  • Transaction filtering by category and date
  • Chart.js integration to visualize spending
  • Dark mode toggle with theme memory
