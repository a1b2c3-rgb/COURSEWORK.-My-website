Weather App

The Weather App is a responsive web application that displays current weather data for any location. It utilizes the OpenWeatherMap API to fetch real-time weather updates based on the user's search.

Features:

  • Real-time temperature, humidity, and wind data
  • Search functionality with autocomplete
  • Responsive design for mobile and desktop

Technologies Used:

HTML, CSS, JavaScript, OpenWeatherMap API

What I Learned:

This project helped me understand how APIs work, how to handle JSON data, and how to make an app user-friendly and responsive.
