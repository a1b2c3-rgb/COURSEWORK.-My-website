Data Dashboard

The Data Dashboard is an interactive data visualization tool that allows users to analyze business metrics over time. It includes charts, tables, and filters to help interpret trends effectively.

Features:

  • Line, bar, and pie charts for visual insights
  • Date range filters and dynamic updates
  • Clean UI with sortable data tables

Technologies Used:

HTML, CSS, JavaScript, Chart.js, Google Sheets API

What I Learned:

This project enhanced my skills in data visualization and taught me how to fetch and manipulate external datasets.
