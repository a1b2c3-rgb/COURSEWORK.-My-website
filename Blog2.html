Building a Personal Finance Tracker

Posted on March 22, 2025 by Violet Talemwa

I chose to address a real-world issue—personal financial management—for one of my portfolio projects.
My goal was to create a straightforward yet effective financial tracker
that would assist users in maintaining control over their earnings and outlays.

How I Built It

I utilized JavaScript to manage user input, balance updates,
and integrate local data, plus HTML and CSS to organize and style the application.
Because all information is stored locally in the browser, users can access it without logging in.

Extra Features

  • Transaction filtering by category and date
  • Chart.js integration to visualize spending
  • Dark mode toggle with theme memory

Lessons Learned

I learned how to manage user mistakes, work with local storage efficiently,
and deconstruct a larger feature into smaller parts from this project.
I also used a charting library for the first time, which was a really educational experience!
